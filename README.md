# 📚 BiblioTrack API

Backend API for the BiblioTrack application.

## Overview
This API handles book management, reservations, and user roles.

## Tech Stack
- .NET Web API
- Microsoft SQL Server

## Features
- CRUD operations for books
- Reservation system
- Role-based authorization (Admin / Visitor)

## API Endpoints
- GET /api/books
- POST /api/books
- etc...

## Run Locally
1. Configure SQL Server
2. Update connection string
3. Run the API
