# 📚BiblioTrack App

A full-stack library management system that allows users to browse, reserve, and manage books online, while providing administrators with tools to manage the library inventory.

Built with .NET, Vue.js, and Microsoft SQL Server.

## ✨ Features

###  Visitor
- Browse and explore available books  
- Reserve books online (ready for pickup after 2 hours)  
- Manage borrowed books  
- Add books to favorites  

###  Admin
- Add, edit, and delete books  
- Manage library inventory  

###  General
- Search, filter, and pagination  
- Role-based access (Admin / Visitor)  
- RESTful API built with .NET  
- Responsive UI built with Vue.js  

---

## Tech Stack

- **Backend:** .NET Web API  
- **Frontend:** Vue.js + Pinia  
- **Database:** Microsoft SQL Server  
- **Hosting:** Render + Azure SQL  

---

## Architecture

Frontend (Vue.js) → Backend (.NET API) → Azure SQL Database  

---

##  Run Locally

1. Clone the repository  
2. Set up a SQL Server database  
3. Update the connection string in `appsettings.json`  
4. Run the backend API  
5. Start the frontend app
